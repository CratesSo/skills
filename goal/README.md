# goal <img src="./assets/icon.png" width="40" alt="goal icon">
- Designs durable Codex self-goals with explicit success and stop conditions.
- Interrogates long-running work until the objective contract is clear enough to start.

## How to use
1. Invoke this skill when a task should become a long-running Codex goal.
2. Answer the depth and intent questions, then review the generated proposed plan before implementation.

### Examples
- `Use goal to turn this migration into a self-goal`
- `Use the goal skill for this long-running cleanup project`

## Version

Current version: v1.0.1

## Install
`npx skills@latest add CratesSo/skills/goal`
