# todo

**Manage a repo-root `todo.md` with fast natural-language add, remove, edit, and list operations.**

**Keeps stable four-character item hashes so todos can survive renumbering.**

## How to use

1. Type `$todo <request>`.
2. Refer to active todos by current number or durable hash.
3. Use natural language for quick changes.

### Examples

- `$todo add review the deploy checklist`
- `$todo add update README and remove stale screenshots`
- `$todo edit 2 to update the API notes`
- `$todo remove a1b2`
- `$todo show`
