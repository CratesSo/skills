# fix <img src="./assets/icon.png" width="40" alt="fix icon">
- Investigate technical failures before proposing or applying fixes.
- Trace root cause, compare working paths, and verify the narrowest useful fix.

## How to use
1. Invoke this skill when debugging bugs, failed tests, build failures, regressions, or unexpected behavior.
2. Follow the evidence-first phases before changing code, then verify the smallest reproduction passes.

### Examples
- `Use the fix skill to debug this failing test`
- `Use fix for this production regression`
