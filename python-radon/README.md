# python-radon
- Run Python complexity and maintainability scans with Radon.
- Reports prioritized hotspots from `radon cc` and `radon mi` before any refactoring.

## How to use
1. Type `$python-radon <scope>`.
2. Review the ranked complexity and maintainability findings before choosing follow-up work.

### Examples
- `$python-radon scan this repo`
- `$python-radon check the services package`

## Version

Current version: v1.0.0

## Install
`npx skills@latest add CratesSo/skills/python-radon`
