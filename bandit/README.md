# bandit
- Run a read-only Python security scan with `bandit`.
- Review findings before acting, and keep results ordered by severity, confidence, and likely impact.

## How to use
1. Type `$bandit <scope>`.
2. Follow the read-only scan and review flow before deciding on any remediation.
3. Stay safe.

## Install
`npx skills@latest add CratesSo/skills/bandit`
