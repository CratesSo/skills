# agents-doctor
- Audit a repo `AGENTS.md` for duplicate, stale, verbose, or low-value guidance.
- Report safe cleanup opportunities first, then apply only confirmed safe changes.

## How to use
1. Invoke the skill from the active repo that contains `AGENTS.md`.
2. Review the audit report, then confirm whether to apply the listed safe changes.

### Examples
- `$agents-doctor`
- `Use the agents-doctor skill to audit this repo's AGENTS.md`

## Version

Current version: v1.0.3

## Install
`npx skills@latest add CratesSo/skills/agents-doctor`
